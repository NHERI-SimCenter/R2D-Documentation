Residual Demand Traffic Simulator
---------------------------------

This tool dialog provides a method to run residual demand traffic simulation for an existing R2D analysis results. The input parameters are the same as the ones used in the R2D :ref:`SP: Residual Demand Traffic Simulator <lbl-SPResidualDemand>`, with one additional input for R2D Results directory. This directory should be defined as the "Results" folder inside the "tmp.SimCenter" folder that can be found in the R2D local working directory. The simulation results will be saved in the "ResidualDemand" directory next to the "tmp.SimCenter" folder.

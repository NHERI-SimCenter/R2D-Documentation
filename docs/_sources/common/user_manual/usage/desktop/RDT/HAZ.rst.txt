HAZ: Hazards
============

In this panel the user can define hazards over a region. The user can select the type of hazard, Earthquakes, Wind events, etc., from the hazard selection combo box, as shown on the top of :numref:`fig-EQSSPanel`. As the user selects between different hazards, the main panel changes to reflect the inputs for each type of hazard application.
